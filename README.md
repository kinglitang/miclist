# miclist

"半吊子" 是一个记录 KTV 歌曲熟练度的静态小站：
- 新坑 / 半吊子 / 绝活 三档状态
- 本地缓存 + CloudBase 云端同步
- Apple Music/iTunes 搜索接入

## 在线访问
- CloudBase：`https://ktv-0g3hatgjf68b912d-1328339263.tcloudbaseapp.com/ktvai2/`
- GitHub Pages：开启后会在 `https://kinglitang.github.io/miclist/`

## 本地使用
直接打开 `ktvai/index.html` 即可。

## 目录结构
- `ktvai/` 静态站点源码
